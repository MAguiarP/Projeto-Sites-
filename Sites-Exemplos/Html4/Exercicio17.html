HTML5

A linguagem HTML(sigla da HyperText Markup Language - Linguagem de Marcação de Hipertexto) é responsável pela criação de documentos para a web, as chamadas páginas. Elas são divulgadas na WWW(World Wide Web)-Teia de Alcance Mundial.

Por meio dessa linguagem, você pode especificar atributos para um texto(como fonte, tamanho, cor, etc..), e fazer os Hyperlinks que permitem navegar pela web. Para isso, ela utiliza marcações, chamadas tags, no ponto da página em que é feito uma ligação com outra.

Portanto, o HTML é um conjunto de tags utilizadas para edição de uma página de Internet. Tudo isso, feito através de uma linguagem fácil e simples de se trabalhar.
